## Project Overview

This project consists of integrating a fully responsive web page based on a provided design mockup.

The objective was to:

- Respect the visual design precisely
- Follow W3C standards
- Optimize the page for SEO
- Implement adaptive images depending on screen size
- Use HTML5, CSS3
- Apply the BEM (Block, Element, Modifier) methodology

The website is fully responsive and works on desktop, tablet, and mobile devices.

---

## Technical Requirements Implemented

### 1. Responsive Design

The layout adapts to different screen sizes using:

- Media queries
- Flexible grids
- Relative units
- Adaptive layout logic

The website can be resized dynamically, and the layout adjusts accordingly.

---

### 2. Adaptive Images

Images are displayed at 100% width and served in different resolutions depending on screen size.

When the browser is resized, the URL of the image changes to load the appropriate resolution. This improves:

- Performance
- Loading speed
- User experience
- SEO ranking

---

### 3. Clean HTML5 Structure

The HTML follows semantic best practices:

- Proper use of `<header>`, `<section>`, `<nav>`, `<footer>`
- Structured headings hierarchy
- Accessibility considerations
- SEO-oriented markup

The structure is logical and optimized for search engine indexing.

---

### 4. CSS3 and BEM Methodology

The project strictly follows the BEM methodology:

- Clear separation between blocks, elements, and modifiers
- Predictable and maintainable class naming
- Scalable architecture

This improves:

- Code readability
- Team collaboration
- Long-term maintainability
